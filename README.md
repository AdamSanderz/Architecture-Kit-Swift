# Architecture Kit
Template to help developer create new component architecture in MVP, MVVM, VIPER, VIVMER, Redux, and any more

## How to install

### Using script
Only need execute this command in terminal:
```swift
sudo swift install.swift
```

### Architecture Template

- [x] MVP
- [x] MVVM
- [ ] MVVM-C
- [ ] VIPER
- [ ] VIVMER
- [ ] TEA
- [ ] REDUX
